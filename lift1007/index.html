LIFT-1007: Learning Technologies

In this module we will explore integrating learning technology tools into classroom activities for pedagogical purposes. This module does not focus on specific tools, instead it is about the larger conversation of using technology effectively, efficiently, and with purpose.

Final Assessment: Educational Technology Review

Select an educational technology software, then produce a description and analysis.

Description

Write a brief description of the tool and what it would be used for.

  • Supported by Saskatchewan Polytechnic?
  • Main features
  • Benefits to learners
  • Limitations
  • Comment on how the technology can be incorporated into classroom activities
  • Provide at least one example of how the product can be used within the classroom.

Analysis

Please include these 4 categories:

  • Universal Design for Learning
  • Is the software safe for students?
  • Ethical
