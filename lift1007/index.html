Microsoft Forms

Saskatchewan Polytechnic is a "microsoft shop" at its core. Students and Faculty are provided with educational Microsoft 365 accounts and many other applications require single sign-on using these accounts. Yes many people choose to use alternatives like Google Docs, Survey Monkey, etc. which reduce security and spread the internal technology ecosystem thin. Why not just learn to use the software we already have access to!?

Description

Microsoft Forms allows users to create surveys and quizzes. They can be entirely anonymous, or record user information. There are many competing software solutions, but Google Forms is the most notable.
