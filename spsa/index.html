SPSA Saskatoon Events

The events below come from the SPSA Event's page under the Saskatoon Events accordion. But it takes several clicks to get there. Luckily their site runs on WordPress, which means it has a built in REST API through /wp-json/ endpoint. We can get their posts from /wp-json/wp/v2/posts/ and filter them using ?categories=15 which returns only "Saskatoon Events".
